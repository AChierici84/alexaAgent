# Agent Alexa-like
## Multiagente con supervisore

Tramite lang-graph verrà implemtato un sistema multiagent alexa like che risponderà ai seguenti compiti:
1. Invocazione meteo tramite scaricamento della pagina associata alla località Richiesta.
2. Oroscopo 
3. Ricerca video su you tube
4. Calendario
5. Calcolatrice
6. Traduttore

Saranno implementati tre agenti: 
- Meteo
- Ricerca You Tube
- Funzionalità Base: quest'ultimo agente avrà a disposizione tool per:
    - Oroscopo
    - Calendario
    - Calcolatrice
    - Traduttore
